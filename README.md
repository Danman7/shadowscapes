# Shadowscapes

A Bun-powered React, TypeScript, Redux Toolkit, Storybook, Tailwind CSS, and ESLint scaffold for a two-player card game UI.

Storybook is the primary surface while the app has no real game logic yet.

## Scripts

- `bun run dev` starts the minimal Vite app shell.
- `bun run storybook` starts Storybook.
- `bun run build-storybook` builds Storybook.
- `bun run typecheck` runs TypeScript checks.
- `bun run eslint` runs ESLint.
- `bun run lint` runs TypeScript and ESLint checks.
- `bun test` runs Bun tests.
